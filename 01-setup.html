NodeJs Project Tutorial

1. Project Setup

Tutorial: Video

  1. Prelims
    1. Install Visual Studio Code (Visual Studio Code)
    2. Install Node.js (NodeJs Website)
    3. Create Root Project Directory
      • Windows Explorer -> New folder
  2. Create Project
    • Open VSC
    • Open Terminal
    • Navigate to project directory, e.g., C:\Users\cni-mark\Projects\[project name]
    • > npm init -y
  3. Install Dependency Packages
    • Basic Packages
      • bcyrpt
      • cookie-parser
      • dotenv
      • ejs
      • express
      • express-ejs-layouts
      • express-session
      • jsonwebtoken
      • method-override
    • To install packages:
      • > npm i --save bcyrpt cookie-parser dotenv express ejs express-ejs-layouts express-session jsonwebtoken method-override
    • Install Nodemon Package
      • > npm i nodemon --save-dev
    • Packages may be installed during project development
  4. Create .env File
    1. In vsc, create new file at project root, .env
  5. Set Up .gitignore
    1. In vsc, create new file at project root, .gitignore
    2. Type:

      node_modules/
      .env

  6. Set Up package.json
    1. In vsc, open package.json
    2. In scripts tag, at end of "test" line, add:

      ,
      "start": "app.js",
      "dev": "nodemon app.js"
