NodeJs Project Tutorial

2. App.js

Tutorial: Video

  1. In vsc root folder, create new folder, "public"
    • Under "public" folder, create 3 new folders, "css", "img", "js">
  2. In vsc root folder, create new file, app.js
  3. Require dependencies
    In app.js type: require('dotenv'.config();
    type: const express = require('express');
  4. Create application
    type:

    const app = express();

    const PORT = 5000 || process.env.PORT;
  5. Add templating engine
    type:
    app.use(expressLayout);
  6. Create intial route
    type:

    app.get('', (req, res) => {

     res.send("Hello World");

    });
  7. Listen to port:
    type:

    app.listen(PORT, () => {

     console.log(`App listening on port ${PORT}`);

    });
  8. Run application
    • In terminal, enter npm run dev
  9. View application
    • In browser, navigate to https://localhost:5000
    • In terminal window, find text "App listening on port 5000
